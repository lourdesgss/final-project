\section{Introduction and Scope}

The goal of this report is to present a detailed description of the current state of the FESTO Factory, making a thorough inventory of its hardware, software, communication protocols, and potential security pitfalls. This was realised through physical inspection of the factory, network capturing, and revision of existing literature detailing functions and protocols. 

This report's scope covers the main security concerns found in the analysis of the factory, backed up by international security standards, as well as proposed mitigation strategies.
